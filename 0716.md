# Git 명령어

## 초기화

`git init`

## File Staging

`git add .`

## Commit 기록하기

`git commit -m '메세지'`

## 원격저장소에 업로드하기(반영하기)

`git push origin main`

## 자주쓰는것 정의하기

<pre>
git config --global alias.ac '!git add . && git commit -m'
git config --global alias.acp '!git add . && git commit -m "update" && git push origin main"
</pre>
